### Как это собрать и запустить
1. Скачать и установить fpc-3.2.2.win32.and.win64.exe отсюда: https://sourceforge.net/projects/freepascal/files/Win32/3.2.2/
2. Скачать или клонировать этот репозиторий 
3. Запустить интересующие вас батники (make_le_x64.bat, make_me_x64.bat, make_il_x64.bat, make_binunp_x64.bat, make_model_x64.bat, make_split_x64.bat)
4. Скопировать все .dll из папки DLLs\x64 к exe-шникам
5. Можно запускать
