#Как это собрать и запустить
1. Скачать и установить fpc-3.2.0.i386-win32.exe отсюда: https://sourceforge.net/projects/freepascal/files/Win32/3.2.0/
2. Скачать и установить fpc-3.2.0.i386-win32.cross.x86_64-win64.exe оттуда же поверх в ту же папку
3. Скачать этот репозиторий 
4. Запустить интересующие вас батники (make_le_x64.bat, make_me_x64.bat, make_binunp_x64.bat, make_model_x64.bat, make_split_x64.bat)
5. Скопировать все .dll из папки DLLs\x64 к exe-шникам
6. Можно запускать
